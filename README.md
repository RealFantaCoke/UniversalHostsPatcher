## Install

### Windows

On Windows run `UniversalHostsPatcher-Windows.exe`

### Linux

On Linux run `sudo java -jar UniversalHostsPatcher-Other.jar`

### MacOS

On MacOS run `sudo java -jar UniversalHostsPatcher-Other.jar`
